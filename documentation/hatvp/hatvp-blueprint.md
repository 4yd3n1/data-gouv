# HATVP Transparency Portal - Complete Application Blueprint

## 🎯 **Project Overview**

The HATVP Transparency Portal is a French government transparency application that provides public access to financial declarations and judicial records of high-level French officials (ministers, deputies, senators). Built as a full-stack TypeScript application, it serves as a digital transparency tool for French citizens to access official financial data.

**Current Status**: Fully functional with minister data imported and verified. Ready for public deployment.

---

## 🏗️ **Architecture Overview**

### **Tech Stack**
- **Backend**: NestJS (Node.js framework) with TypeScript
- **Frontend**: Next.js 15 with React 18, TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **UI**: Tailwind CSS with shadcn/ui components
- **State Management**: React Query (TanStack Query)
- **API**: RESTful with OpenAPI/Swagger documentation

### **Deployment Configuration**
- **Frontend**: `http://localhost:3000` (Next.js dev server)
- **Backend**: `http://localhost:4000/api/v1` (NestJS production server)
- **Database**: PostgreSQL with Prisma migrations
- **CORS**: Configured for local development and production origins

## 📈 **Financial Calculation Engine**

### **Remuneration Calculations**

#### **Activity Types**
- **Professional Activities**: Main employment income
- **Advisory Activities**: Consulting fees
- **Elective Mandates**: Political position compensation
- **Board Participations**: Corporate board memberships

#### **Calculation Logic**
```typescript
totalRemunerations = sum(
  professional_activities.remunerations +
  advisory_activities.remunerations +
  elective_mandates.remunerations +
  board_participations.remunerations
)
```

#### **Yearly Breakdown**
- Grouped by calendar year
- Activity type categorization
- Trend analysis support

### **Asset Calculations**

#### **Asset Categories**
- **Real Estate**: Properties, land, commercial real estate
- **Bank Accounts**: Checking, savings, investment accounts
- **Life Insurance**: Investment contracts
- **Listed Securities**: Publicly traded stocks/bonds
- **Unlisted Securities**: Private company holdings
- **SCI Holdings**: Real estate investment companies
- **Professional Assets**: Business-related holdings
- **Foreign Assets**: International investments
- **Other Assets**: Miscellaneous holdings
- **Vehicles**: Cars, motorcycles, etc.

#### **Liability Categories**
- **Debts**: Mortgages, loans, credit cards
- **Other Passives**: Legal obligations, guarantees

#### **Net Worth Formula**
```typescript
grossAssets = sum(all_asset_categories)
totalLiabilities = sum(all_liability_categories)
netWorth = grossAssets - totalLiabilities
```

#### **Validation Rules**
- **Non-negative Values**: All monetary amounts ≥ 0
- **Date Consistency**: Declaration dates within reasonable ranges
- **Required Fields**: Critical data presence validation
- **Reference Integrity**: Foreign key relationships maintained

#### **Error Handling**
- **Graceful Degradation**: Missing data doesn't break calculations
- **Logging**: Comprehensive error tracking
- **Fallback Values**: Safe defaults for missing data
